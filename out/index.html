Philosophy

Learn by building, together

This community is built on a simple belief: the best way to learn is by doing real work, getting feedback, and building alongside others who are shipping production systems.

Learning by doing

No passive consumption. Every activity is designed around building, shipping, and getting feedback on real work.

Production-ready

Focus on what actually works in production. Move from prototypes to reliable systems with battle-tested patterns.

Build together

Work alongside other practitioners. Hackathons, projects, and group problem-solving—not isolated learning.

Calibrate your judgment

Develop better instincts through exposure to Alexey's reasoning process and peer feedback on real decisions.

To-Do List Agent

by Alexey Grigorevintermediate

A reference project from the AI Bootcamp—an agent that interacts with a simple to-do list application. Built with Lovable for the frontend and FastAPI (Python) for the backend. Uses the backend's OpenAPI spec so the model can create tools to get tasks or mark them complete, with Logfire for monitoring and pytest for testing.
