d:T609,<p>An <strong>AI-powered data cleaning assistant</strong> that accepts CSV or Excel via web UI or API, analyzes dataset structure and data quality issues, generates cleaning suggestions using intelligent detection, and applies transformations deterministically. You get a cleaned dataset plus a human-readable report of what changed.</p>
<p><strong>Why it's useful:</strong> Raw tabular data is often messy—inconsistent column names, missing values, mixed types, duplicates. This project turns that into a <strong>repeatable pipeline</strong>: the LLM acts as a <strong>planner</strong> (suggesting what to do), while the <strong>cleaning engine</strong> is deterministic and testable. The repo includes unit tests, integration tests, E2E tests (Playwright), OpenAPI-first design, Docker, and deployment to <strong>Google Cloud Run</strong>.</p>
<p><strong>Features:</strong> Column standardization (snake_case), non-value detection (ERROR/UNKNOWN → NaN), string normalization, smart type casting (numeric, datetime), duplicate removal, null handling, ID-column protection. The system uses MCP-style boundaries: structured JSON in/out between the LLM and the execution layer.</p>
<p><strong>Tech stack:</strong> FastAPI (Python), React + TypeScript, OpenAI-compatible LLM, Pandas, SQLite/Postgres, OpenAPI, Docker, GitHub Actions, GCP Cloud Run.</p>
