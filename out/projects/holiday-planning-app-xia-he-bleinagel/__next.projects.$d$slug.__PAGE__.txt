<p><strong>Xia He-Bleinagel</strong> (Head of Data &#x26; Cloud) built a comprehensive <strong>holiday planning app</strong> to replace endless Excel sheets for managing team holidays.</p>
<p>The app is built on <strong>Django</strong> and uses a mix of AI tools in the workflow:</p>
<ul>
<li><strong>ChatGPT</strong> — to refine prompts (e.g. by uploading a design mock and improving instructions)</li>
<li><strong>Lovable</strong> — to create the UI and frontend (cleaner output than Cursor for design)</li>
<li><strong>Cursor</strong> — for implementation (ran into repeated bugs, fixed 6–7 times)</li>

</ul>
<p><strong>Features:</strong> Supabase Postgres, secure authentication with admin-invited users, different UI and permissions per role, and integrated <strong>school holidays</strong> and <strong>German national holidays</strong>—no more static spreadsheets per year.</p>
