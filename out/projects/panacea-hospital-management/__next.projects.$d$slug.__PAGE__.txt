c:T66d,<p><strong>Panacea</strong> is a modern, full-stack hospital management platform for streamlined patient care, appointment scheduling, and clinical operations.</p>
<p><strong>Why it's useful:</strong> It doubles as a <strong>reference implementation</strong> for building <strong>sensitive-data systems</strong>. Production-grade structure is built in from the start:</p>
<ul>
<li><strong>Role-based access control (RBAC)</strong> — Admin, Doctor, Nurse, Receptionist, Patient roles with strict endpoint protection</li>
<li><strong>Audit logging</strong> — Major actions logged for compliance (AuditLog collection)</li>
<li><strong>Input validation</strong> — Mongoose schemas plus centralized validators</li>
<li><strong>Rate limiting</strong> — Brute-force and DoS protection</li>
<li><strong>Soft deletes</strong> — Critical entities (e.g. Patient) use <code>isDeleted</code> with transparent filtering so data is never hard-deleted</li>
<li><strong>Security</strong> — Helmet, CORS, HPP, consistent API response shapes</li>
</ul>
<p><strong>Core features:</strong> Landing page, authentication, patient registry with medical history, appointments (schedule and status), medical records, ward management (bed occupancy), analytics dashboard.</p>
<p><strong>Tech stack:</strong> React 19 + Vite, TanStack Query, React Router, Tailwind + shadcn/ui, Recharts; Node.js + Express 5, MongoDB + Mongoose, JWT, Winston + Morgan. Docker Compose for production-like runs.</p>
