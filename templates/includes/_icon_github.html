{% comment %} Inline GitHub mark SVG (issue #518). Replaces ```` placeholders that no longer hydrate because upstream Lucide moved brand icons out of the core package — the empty ```` element kept its ``h-4 w-4`` slot plus the parent's ``gap-2``, producing a phantom left-padding and a missing icon in dark mode. Render with ``{% include 'includes/_icon_github.html' with css='h-4 w-4' %}``. The ``css`` parameter sets the wrapper SVG's class so callers control size and alignment exactly like they did with the Lucide ````. The fill uses ``currentColor`` so the icon inherits the foreground color in both light and dark themes. The path comes from the Lucide ``github`` glyph (CC-BY-4.0 / ISC) drawn with ``stroke="currentColor"`` to match the visual weight of sibling ``data-lucide`` icons (``external-link``, ``book-open``). {% endcomment %}
