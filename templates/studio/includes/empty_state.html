{% comment %} Canonical Studio list-page empty state. Two flavours: - ``kind="filter"`` — a search or status filter is active and produced zero rows. Renders an inline ```` row so the table header stays visible while the operator adjusts the filter. - ``kind="fresh"`` — the entity has no rows at all. Renders a separate ``bg-card`` empty card (no table chrome) with a primary CTA so the page looks inviting on a fresh install. Context: kind -- "filter" | "fresh" entity_label -- singular, lowercase ("sprint", "campaign") entity_label_plural -- plural, lowercase ("sprints", "campaigns") create_url -- target for the "New " CTA (fresh only). Default 8. cta_label -- optional override for the CTA label. Defaults to "New ". testid_suffix -- optional extra ``data-testid`` to keep page-specific selectors working alongside the canonical testid.
